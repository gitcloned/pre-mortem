Full User Journey

Three phases — each independently shippable

PHASE 1

Auth + Organisation

Landing page
→
Sign up / Login
→
Create Organisation
→
Invite members (link or email)
→
Org dashboard
Roles: Owner (creates org) · Admin (manages members) · Member (creates projects)
PHASE 2

Project Management

Org dashboard
→
Create project
→
Project page
→
Archive / Reactivate
Each project holds multiple pre-mortem sessions over its lifetime
PHASE 3 — CORE

Pre-mortem Session

Create session
→
Share link
→
Team adds risks (private)
→
Facilitator reveals all
→
Team votes (3 votes each)
→
Assign action items
→
Export
Risk input per item
Title · Description · Category
(timeline / technical / team / external / scope / comms)
Likelihood (1–5) · Impact (1–5)
Session states
Draft → Open (input) → Revealed → Voting → Closed
Facilitator controls each transition
